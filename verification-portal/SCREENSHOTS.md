## Key Features

1. **Drag and Drop**: Simple file upload interface
2. **Instant Verification**: Results in < 1 second
3. **Visual Feedback**: Green for success, red for failure
4. **Detailed Checks**: Shows exactly what passed/failed
5. **Error Reporting**: Clear messages about what went wrong
6. **Certificate Details**: Shows job metadata
7. **No Technical Knowledge**: Designed for auditors and compliance teams

## Tech Stack

- **Backend**: Express.js + Multer + Unzipper
- **Frontend**: Vanilla HTML/CSS/JavaScript (no build required)
- **Crypto**: Native Node.js crypto module
- **Deployment**: Single command (`npm start`)
